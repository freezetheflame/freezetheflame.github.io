NP问题：

P：所有问题多项式时间可解

NP：在多项式时间内“可验证”的问题。

NPC：存在这样一个NP问题，所有的NP问题都可以约化成它。

NP-complete：NP范围且和NP内任何一个问题一样难的问题

Interval Scheduling

这一题的大意就是要安排一个最多，但是时间互不矛盾的时间表。

要解决这么一个题目，首先明确采用贪心的方式解决

需要设计一个求解局部最优且不影响后效的贪心策略（贪心算法的无后效性）
